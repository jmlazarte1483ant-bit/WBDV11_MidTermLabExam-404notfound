# WBDV11_MidTermLabExam-404notfound
The project consists of creating a simple multi-page website using HTML and CSS.  
It focuses on building a clean layout, navigation, and basic design.

This website is about a milk tea shop that offers affordable drinks for students.  
We also added a promo feature to make the website more realistic.

Some parts of the project involved trial and error while designing and arranging elements, but overall the website works properly.
